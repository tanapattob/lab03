Bootstrap :: การใช้งาน form

Post by Tanapat, Feb 07, 2017.
Bootstrap form

Bootstrap layout แบบ form


Form ของ Bootstrap ซึ่งก็ไม่ได้มีอะไรซับซ้อนมากมายครับ โดยเราแค่เรียกใช้ class form-control ใน textbox combo-box หรือ textarea แค่นี้ input ต่างๆ ก็แสดงผลในรูปแบบของ bootstrap ครับ โดย input ต่างๆ ที่มีการเรียกใช้ class form-control จะมีลักษณะที่โค้งมน และมีความกว้าง 100% ใน tag ที่ input นี้อยู่ครับ
เรามาดู From แบบแรกกันเลยครับ

Inline From

ก็ตามชื่อครับ เป็น from ที่มีพวก input อยู่ในแถวเดียวกัน ซึ่งหน้าตาก็จะประมาณนี้ครับ

ซึ่งเบื้องต้นก็นำ โค๊ดด้านล่างไปวางไว้ในหน้าที่มีการเรียกใช้งาน bootstrap อยู่แล้วก็สามารถใช้งานได้เลยครับ

แบบที่สองครับ

Horizontal form

ตัวนี้จะเป็นฟอร์มตามแนวนอนครับ โดย input ต่างๆ ปกติ จะอยู่คนละแถวกันครับ จะมีอยู่แถวเดียวกันก็เพียง label เท่านั้น ซึ่ง Horizontal form นั้นมีการนำ grid system ที่เราได้กล่าวไปแล้วใน บทความที่แล้ว เรื่อง contanier และ grid ครับ ซึ่งหน้าตาของ Horizontal form ก็จะประมาณนี้ครับ




ก่อนจะจบบทความนี้ของทิ้งท้ายนิดนึงครับ สำหรับมือใหม่ทั้งหลาย อย่าลืม เรียกใช้ class เพื่อกำหนดรูปแบบ ของ form ใน form และ กำหนด class ใน input ต่างๆ ใน form เพื่อการแสดงผลที่ถูกต้องครับ
